# Employee Tracker

## Description

A MySQL and Node application that allows you to add Departments, Roles, and Employees for easy tracking of your workforce!

## Table of Contents

- [Installation](#installation)
- [Usage](#usage)
- [Contribution Guidelines](#contributing)
- [Testing](#testing)

## Installation

1. Fork or download this repository.
2. Make sure you have node & MySQL installed on your local device.
3. Run `npm i` or `npm install` to ensure all dependencaies are on your local device
4. Update db/connection.js dummy text with your MySQL user and password
5. Set up MySQL (SOURCE db/db.sql, USE employee_tracker, SOURCE db/schema.sql, \*\*SOURCE db/seeds.db if you want to use dummy data)

## Usage

**After installation**:

1. Run `node server.js`
2. Answer prompts
3. Done!
